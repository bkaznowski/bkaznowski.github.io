This is the first post on the blog. It's built with [Jekyll](https://jekyllrb.com/) and hosted for free on [GitHub Pages](https://pages.github.com/).

Two kinds of posts will show up here:

- **Tech** — notes on code, tools, and things I'm building.
- **Running** — training logs, races, and gear notes.

To write a new post, add a Markdown file to `_posts/` named `YYYY-MM-DD-title.md` with a front-matter block like this one at the top, then push to GitHub. No build step required — GitHub Pages renders it automatically.
